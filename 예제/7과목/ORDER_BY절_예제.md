### 성적 테이블

| 이름 | 과목 | 학점 |
| --- | --- | --- |
| 김철수 | C언어 | A |
| 한유리 | 자료구조 | A |
| 신짱구 | 자료구조 | A |
| 이훈이 | 알고리즘 | B |
<br />

#### 1. 성적 테이블에서 이름에 대해 오름차순으로 정렬하여 모두 출력

```sql
SELECT *
FROM 성적
ORDER BY 이름;
```
<br />

#### 2. 성적 테이블에서 과목에 대해 오름차순으로 정렬 후 같은 값인 경우 이름에 대해 오름차순으로 정렬하여 모두 출력

```sql
SELECT *
FROM 성적
ORDER BY 과목, 이름;
```
<br />

#### 3. 성적 테이블에서 학점은 내림차순, 이름은 오름차순 정렬하여 모두 출력

```sql
SELECT *
FROM 성적
ORDER BY 학점 DESC, 이름 ASC;
```
